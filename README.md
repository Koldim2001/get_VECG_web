# Вэб версия проекта по ВЭКГ (только под запуск через python)

Эта версия имеет подход R-R интервалов и позволяет выбирать число страниц вывода. <br/>

То есть можно указать вывод на 1 странице всех графиков или для каждого plotly графика свою страницу открывать (с указанием светлой или темной темы отображения)<br/>

К сожалению, когда plotly пытается показать график с помощью fig.show(), он открывает новую вкладку браузера с локальным адресом (127.0.0.1) и случайным портом, который может не быть доступным извне контейнера Docker. Так что такой подход хоть внешне и прикольный, но для финальной серверной версии не применим. <br/>

По этой причине я и сохранил эту версию кода в отдельной ветке. Для запуска надо в терминале написать: streamlit run main.py
